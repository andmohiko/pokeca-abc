# V1 Step3: スコア化と実装 — ソウブレイズexデッキ

## 実装方針

サンプルコード（Dragapult ex / Mega Lucario ex）のアーキテクチャを踏襲し、ソウブレイズexデッキの戦略をスコアリングロジックに変換した。

### 基本アーキテクチャ

```
agent(obs_dict)
  → Observationに変換
  → select.option の各選択肢にスコアを付ける
  → スコア降順で maxCount 個を返す
```

## スコア設計

### ターン内の行動優先度（スコア帯）

スコアは「行動の強さ」ではなく「ターン内での実行順序の優先度」を表す。
手札に行動できるカードがあるかどうかでドローソースのスコアが変動する。

| スコア帯 | 行動 | 理由 |
|---|---|---|
| 110,000 | ディスカード時の闘エネルギー | トラッシュに送るのが最善。炎エネより優先 |
| 100,000 | ディスカード時の炎エネルギー/スボミー選択（後1） | しんえんほむらの火力に直結 |
| 95,000 | パーフェクトミキサー | **手札にあるなら最初に打つ。** 山札から5枚までトラッシュ |
| 92,000 | アタッカー入れ替え用の手貼り（ベンチにエネ付きソウブレイズexがいる時） | 育ったアタッカーを前に出すための手貼り |
| 91,000 | アタッカー入れ替え用のにげる | 手貼り後ににげてソウブレイズexを前に出す |
| 90,000 | 炎エネルギー手貼り（バトル場ソウブレイズex） | どの行動よりも先に手貼り |
| 88,000 | 炎エネルギー手貼り（バトル場カルボウ） | ちからをあつめるで基本エネ2枚サーチ |
| 86,000 | 炎エネルギー手貼り（ベンチソウブレイズex） | 予備アタッカーの攻撃準備 |
| 85,000 | 後1のエネルギー手貼り（にげるコスト用） | スボミーをバトル場に出すための手貼り |
| 84,000 | 炎エネルギー手貼り（ベンチカルボウ） | 進化後に即攻撃可能にする布石 |
| 82,000 | 後1のにげる（スボミーをバトル場に） | 手貼り後ににげてスボミーを前に出す |
| 75,000 | さかてにとる（きぜつ直後） | ドローソース最優先。サポート権を消費しない |
| 70,000 | 進化（カルボウ→ソウブレイズex） | 進化を先にすると後続の行動を制御しやすい |
| 60,000 | ほりまくり（モグリューの特性） | ベンチに出した時に闘エネを最大3枚トラッシュ |
| 53,000 | キチキギスexをベンチに出す | 手札にあれば積極的に出す。きぜつ後のドロー特性に備える |
| 51,000〜55,000 | ポケモンをベンチに出す | 盤面構築は早めに |
| 49,000 | なかよしポフィン | カルボウ展開。イワパレスモード時はオーガポン最優先 |
| 47,000〜48,000 | ファイトゴング | モグリュー/闘エネサーチ |
| 46,000 | ポケパッド | カルボウ・モグリューサーチ |
| 45,000 | ハイパーボール | コストでエネをトラッシュ。ソウブレイズex/キチキギスexサーチ |
| 44,000 | 夜のタンカ | 状況により回収対象が変わる（後述） |
| 35,000 | ボスの指令 | 相手exをワンパンできる場合。対イワパレス時はイシズマイ呼び出し |
| 14,000 | ドロー系サポート（行動可能なカードがある時） | 手札を使い切ってから |
| 10,000 | にげる（通常） | 必要な場合のみ |
| attackId | 攻撃（しんえんほむら等） | ターン最後の行動 |
| -1 | 不要な行動 | スキップ可能なら選択しない |
| -10,000,000 | 絶対に不要 | 完全に無駄な行動 |

### パーフェクトミキサー

ACE SPEC。山札から好きなカードを5枚までトラッシュ。**山札に闘エネルギーがある場合のみ使う（スコア95,000）。闘エネがなければ使わない（-1）。**

#### パーフェクトミキサーでトラッシュするカードの選択（TO_HANDコンテキスト）

パーフェクトミキサー使用時、山札のカードを選ぶ際のスコア。**スコアが-1のカードは選択しない**（出力ロジックでパーフェクトミキサー選択時は負スコアをスキップ）。

**通常時:**

| 優先 | カード | 条件 | スコア | 理由 |
|---|---|---|---|---|
| 1 | 闘エネルギー | 常に | 100,000 | できるだけ多くトラッシュ |
| 2 | スボミー | 後攻1ターン目以外 | 80,000 | 使わないカードを山札から除去 |
| 3 | オーガポン | イワパレスではない確定時 | 80,000 | 不要カード除去 |
| 4 | 炎エネルギー | 場の炎エネが3枚以上 | 70,000 | 攻撃コスト分は場で確保済み |
| - | その他すべて | - | -1 | **トラッシュしない** |

**LO危険時（`lo_danger or ideal_board`）:**

| カード | スコア | 理由 |
|---|---|---|
| 闘エネルギー | 100,000 | 唯一のトラッシュ対象 |
| その他すべて | -1 | LOリスク軽減のため一切捨てない |

※ 「場の炎エネが3枚以上」= バトル場+ベンチのポケモンに付いている炎エネルギーの合計。手札は含めない。

### アタッカー入れ替え

バトル場のポケモンがソウブレイズex以外で、ベンチにエネルギー付きソウブレイズexがいる場合：
バトル場のポケモンにエネルギーを手貼り（スコア92,000）→ にげる（スコア91,000）→ ソウブレイズexを前に出す。

**パーフェクトミキサー・炎エネ手貼りに次ぐ高優先度で行う。**

### キチキギスex退避

キチキギスexがバトル場に出てしまった場合、最優先で逃がす。

```python
do_retreat_for_fezandipiti = (active_id == Fezandipiti_ex
                              and not do_retreat_for_budew
                              and not do_retreat_for_ogerpon
                              and not do_retreat_for_attacker
                              and len(my_state.bench) > 0)
```

| 行動 | スコア | 条件 |
|---|---|---|
| ポケモンいれかえ（PLAY） | 38,000（do_switch経由） | 手札にある場合 |
| エネルギー手貼り（ATTACH） | 92,000 | にげるコスト用 |
| にげる（RETREAT） | 91,000 | 手貼り後に逃げる |

**入れ替え先:** SWITCH/TO_ACTIVEコンテキストで、アタッカー不在時はモグリューを優先（12,000）。カルボウ（10,000）は進化元として温存する。

### ドローソースのスコア制御

手札に行動できるカード（サポート・エネルギー以外のPLAYオプション、EVOLVEオプション）があるかをチェックし、ドロー系のスコアを動的に変更する。

| 状態 | ドロー系サポートのスコア | 挙動 |
|---|---|---|
| 手札に行動できるカードがない | 90,000 | 手貼りの直後、最優先でドロー |
| 手札に行動できるカードがある | 14,000 | 手札を使い切ってからドロー |

※ さかてにとる(75,000)は固定スコア。

### ドローソースの優先度

| 優先 | ドローソース | 種別 | スコア |
|---|---|---|---|
| 1 | さかてにとる（キチキギスex特性） | 特性 | 75,000（きぜつ直後）/ 40,000（通常） |
| 2 | リーリエの決心 | サポート | draw_support_score（通常時優先） |
| 3 | ゼイユ | サポート | draw_support_score（手札にエネが多い時のみリーリエより優先） |

### モグリュー + 闘エネルギーのトラッシュ加速

#### ほりまくり（特性）
モグリューをベンチに出した時に発動。山札から基本闘エネルギーを最大3枚トラッシュ。
ABILITYオプションでスコア60,000を付与。

#### ファイトゴング
モグリューまたは闘エネルギーをサーチ。
- 山札にモグリューがいる＆場に2体未満: スコア47,000
- 山札に闘エネがある: スコア45,000
- それ以外: -1

#### ポケパッド
山札からルールを持たないポケモン（カルボウ・モグリュー・オーガポン）を1枚サーチ。
- 対イワパレス時に山札にオーガポンがいて場にいない: スコア46,000
- 山札にモグリューがいて場に2体未満: スコア46,000
- 山札にカルボウがいてアタッカー3体未満: スコア46,000
- それ以外: -1

#### 闘エネルギーの手貼り
闘エネルギーは攻撃コストに使えないため基本手貼りしない（スコア-1）。
**例外**: 後攻1ターン目のにげるコスト用（85,000）、アタッカー入れ替え用（92,000）。

### ディスカード（コスト支払い時）

ハイパーボールのコストやゼイユの効果でカードをトラッシュする際の優先度：

```
闘エネルギー → スコア110,000（最優先）
炎エネルギー → スコア100,000（場に炎エネ付きポケモンがいるor手札に2枚以上）
             → スコア-50,000（場に炎エネなし＆最後の1枚→残す）
スボミー     → スコア60,000（後1以外。コストにしてよい）
             → スコア-200,000（後1。絶対に捨てない）
不要グッズ   → スコア50,000（序盤以降のポフィン等）
その他グッズ → スコア-10,000
夜のタンカ   → スコア-200,000（対イワパレス時は絶対に捨てない）
サポート     → スコア-200,000（捨てない）
ソウブレイズex/カルボウ → スコア-200,000（捨てない）
オーガポン      → スコア60,000（不要判定時。スボミー同様捨ててよい）
             → スコア-200,000（対イワパレス時/相手が草エネ使用時。絶対に捨てない）
ACE SPEC     → スコア-200,000（捨てない）
```

### エネルギー手貼り

しんえんほむらのコストは炎1個のみ。1体につきエネルギーは1枚だけ付ければよい。すでにエネルギーが付いているポケモンには付けない。

**炎エネルギーの手貼りはパーフェクトミキサーの次に優先して行う。** 攻撃準備を早めに確定させる。

#### 手貼り先の優先度（エネルギーが0枚のポケモンのみ対象）

| 優先 | 対象 | スコア | 理由 |
|---|---|---|---|
| 1 | バトル場のソウブレイズex | 90,000 | 即攻撃可能にする |
| 2 | バトル場のカルボウ | 88,000 | ちからをあつめるで基本エネ2枚サーチ |
| 3 | ベンチのソウブレイズex | 86,000 | 予備アタッカーの攻撃準備 |
| 4 | ベンチのカルボウ | 84,000 | 進化後に即攻撃可能にする布石 |
| - | モグリュー/スボミー/キチキギスex | -1 | 攻撃しないポケモンには付けない |
| - | エネルギーが1枚以上付いているポケモン | -1 | 1体1枚で十分（※オーガポンは例外、3枚まで付ける） |

#### 例外1: 後攻1ターン目
バトル場のポケモンにエネルギーを手貼りし、にげるコストとして使用してスボミーをバトル場に出す（スコア85,000）。

#### 例外2: アタッカー入れ替え
ベンチにエネ付きソウブレイズexがいてバトル場が非アタッカーの場合、バトル場のポケモンに手貼り（スコア92,000）→ にげる（スコア91,000）でソウブレイズexを前に出す。

### サポート選択

1ターンに1枚のみ使えるため、ターン開始時に最適なサポートを事前決定する：

| 優先度 | サポート | 条件 |
|---|---|---|
| 1 | ボスの指令 | 相手ベンチにワンパンできるexがいる、または対イワパレス時にイシズマイ/他ポケモンを呼び出す |
| 2 | ゼイユ | 捨てられるカード > 捨てたくないカード |
| 3 | リーリエの決心 | 通常時のドローソース |
| 4 | ゼイユ | 最終フォールバック。条件が悪くても使わないよりマシ |

#### ゼイユの使用条件（リーリエより優先される条件）

**捨てられるカードの枚数が捨てたくないカードの枚数より多ければ、リーリエより優先する。**

**捨てられるカード:**
- 闘エネルギー
- 炎エネルギー（トラッシュの炎エネが4枚未満の場合のみ）
- スボミー（後攻1ターン目以外）
- オーガポン（不要判定時のみ）

**捨てたくないカード:**
- ソウブレイズex、カルボウ、パーフェクトミキサー（ACE SPEC）
- オーガポン（不要判定でない場合）
- 夜のタンカ（対イワパレス時）
- サポートカード（ゼイユ自身を除く）

```python
discardable = hand_fighting_energy_count
if discard_fire_energy_count < 4:
    discardable += hand_fire_energy_count
if not is_going_second_turn1:
    discardable += hand_budew_count
if ogerpon_unnecessary:
    discardable += hand_counts.get(Ogerpon_Hearth, 0)
# 捨てたくない = ソウブレイズex + カルボウ + ACE SPEC + オーガポン(不要でなければ) + 夜のタンカ(対イワパレス時) + サポート(ゼイユ除く)
zeyu_preferred = discardable > undiscardable
```

条件を満たさなくても、他にサポートがなければ最終フォールバックとしてゼイユを使う。

#### 先攻1ターン目の注意
先攻1ターン目はゼイユのみ使用可能（リーリエ・ボスは使えない）。実装ではselect.optionに実際にプレイ可能なサポートのみが含まれるため、`playable_supporters`を使って判定する。

### サーチ系カードのポケモン選択優先度

#### 通常時（TO_BENCHコンテキスト）

| 優先 | 条件 | ポケモン | スコア |
|---|---|---|---|
| 1 | 後攻1ターン目 | スボミー | 100,000 |
| 2 | アタッカー（カルボウ+ソウブレイズex）が1体以下 | カルボウ | 50,000 |
| 3 | アタッカーが2体で、カルボウが2枚以上 | モグリュー | 40,000 |
| - | キチキギスexが場にいない AND ベンチ4体以上 | 出さない | -1（キチキギスex用の枠を確保） |
| - | アタッカーが3体以上 | 出さない | -1 |
| - | 後攻1ターン目以外 | スボミー | -1（出さない） |

**ベンチ枠予約: キチキギスexが場にいない間は、カルボウ/モグリューでベンチを4体以上にしない。** キチキギスexが手札に来たとき必ず出せるようにする。キチキギスexは必ずベンチに出せるため、ゼイユやハイパーボールのコストにしても問題ない。

**なかよしポフィンのベンチ出し対象制限。** TO_BENCHコンテキストで`select.effect.id == Buddy_Buddy_Poffin`の場合、以下のルールで対象を絞る：

| ポケモン | ポフィン対象 | スコア | 条件 |
|---|---|---|---|
| オーガポン | 対イワパレスモード時のみ | 100,000（最優先） | イワパレス確定時のみ |
| カルボウ | 常に対象 | 50,000 | — |
| スボミー | 後攻1ターン目のみ対象 | 40,000 | それ以外は-1 |
| モグリュー | 常に対象外 | -1 | — |
| その他 | 常に対象外 | -1 | — |

**通常時はカルボウのみ。山札にカルボウがいなければポフィンを使わない。**

#### 後攻1ターン目のポフィン

理想盤面はバトル場にスボミー、ベンチにカルボウ。

| 場の状態 | ポフィンで出すポケモン |
|---|---|
| スボミーがいない | カルボウ + スボミー |
| スボミーがいる | カルボウ2体 |

**スボミーが手札またはベンチにいてバトル場に出せるなら、必ずむずむずかふんで攻撃する。**

#### ポケモンのサーチ優先度（TO_HANDコンテキスト等）

ハイパーボール・夜のタンカ等でポケモンをサーチする際の優先度。場の状況で分岐する：

| 優先 | 条件 | サーチするポケモン | スコア | 理由 |
|---|---|---|---|---|
| 1 | 場にカルボウがいない | カルボウ | 50,000 | 進化元がいなければ次のアタッカーを立てられない |
| 2 | 場にカルボウはいるがソウブレイズexがいない | ソウブレイズex | 45,000 | 2体目のカルボウではなくソウブレイズexを優先 |
| 3 | 場にソウブレイズexが2体いる | キチキギスex | 40,000 | アタッカーは十分。きぜつ後のドロー特性で手札を補充 |
| 4 | キチキギスexも場にいる | モグリュー | 35,000 | ほりまくりでトラッシュ肥やしを加速 |

### 後攻1ターン目のスボミー展開

スボミーをバトル場に出すための特殊シーケンス：

```
1. スボミーをベンチに出す（55,000）
2. バトル場のポケモンにエネルギー手貼り（85,000）
3. にげるでスボミーと入れ替え（82,000）
4. ターン最後にむずむずかふん（attackId）
```

### 夜のタンカの回収優先度

夜のタンカの使用スコアは44,000。TO_HANDコンテキストでの回収対象は、リソース枯渇チェック → アタッカー準備のボトルネックの順で判定する。

**闘エネルギーは回収しない。** ソウブレイズexも炎エネも足りているなら使わない。
**対イワパレス時は夜のタンカをオーガポン回収専用に温存する。** トラッシュにオーガポンがいる場合のみ使用する。

#### 1. リソース枯渇チェック（優先）

| 条件 | 回収対象 | スコア | 理由 |
|---|---|---|---|
| トラッシュに炎エネが5枚以上 | 炎エネルギー | 70,000 | 山札にほぼ残っていない |
| トラッシュにソウブレイズexが3体以上 AND `ceil(相手残りサイド/2) > 場のソウブレイズex数` | ソウブレイズex | 65,000 | 山札に残っていない＆追加が必要 |

#### 2. アタッカー準備のボトルネック（通常）

| パターン | 場の状態 | 回収対象 | スコア |
|---|---|---|---|
| A | 場にカルボウもソウブレイズexもいない | カルボウ | 50,000 |
| B | カルボウはいるがソウブレイズexが手札にも場にもない | ソウブレイズex | 50,000 |
| C | ソウブレイズexはいるが炎エネが付いておらず手札にもない | 炎エネルギー | 60,000 |
| - | ソウブレイズexも炎エネも足りている | 使わない | -1 |

#### モグリューのベンチ出し

- **SETUP_BENCH_POKEMON（バトル開始前）ではモグリューを出さない（スコア-1）**
- 自分の最初の番で手札にモグリューがあれば最優先でベンチに出す（PLAYスコア52,000）

### 状況別のスコア調整

#### 先攻 vs 後攻

- **先攻選択**（IS_FIRSTでYESにスコア1）
- **後攻の場合**: SETUP_ACTIVE_POKEMONでスボミーに最高スコア（100,000）を付与し、グッズロックを狙う
- **後攻1ターン目**: 手貼り→にげる→スボミー展開の特殊シーケンスを発動

#### きぜつ直後（pre_ko = True）

- キチキギスexの「さかてにとる」: スコア75,000（ドローソース最優先。サポート権を消費しない）
- 夜のタンカ: ソウブレイズexの回収スコアが上昇

### LO（ライブラリーアウト）対策

山札を引き切って負けることを防ぐ。

#### 理想盤面判定

```python
ceruledge_with_energy = 0
for card in my_state.active:
    if card is not None and card.id == Ceruledge_ex and len(card.energies) >= 1:
        ceruledge_with_energy += 1
for card in my_state.bench:
    if card.id == Ceruledge_ex and len(card.energies) >= 1:
        ceruledge_with_energy += 1
ideal_board = ceruledge_with_energy >= 2
```

#### ドロー抑制

| 条件 | 制限 | 実装 |
|---|---|---|
| `ideal_board == True` | ドローサポートを使わない | `draw_support_score = -1` |
| `my_state.deckCount <= 8` | ゼイユ・リーリエを使わない | サポート選択でゼイユ・リーリエを除外 |

**ボスの指令は山札を引かないため制限対象外。**

### オーガポン3状態判定

相手のデッキがイワパレスかどうかを3状態で管理する。

```python
# グローバル変数で試合中の状態を保持
_op_confirmed_not_iwaparesu = False

# ターン0でリセット
if state.turn == 0:
    _op_confirmed_not_iwaparesu = False

# 相手が草以外のエネルギーを使っていたら「イワパレスではない」確定（試合中維持）
if not _op_confirmed_not_iwaparesu:
    for card in op_state.active + op_state.bench:
        for ec in card.energyCards:
            if card_table[ec.id].cardType in (CardType.BASIC_ENERGY, CardType.SPECIAL_ENERGY) and ec.id != Basic_Grass_Energy:
                _op_confirmed_not_iwaparesu = True
    for card in op_state.discard:
        if card_table[card.id].cardType in (CardType.BASIC_ENERGY, CardType.SPECIAL_ENERGY) and card.id != Basic_Grass_Energy:
            _op_confirmed_not_iwaparesu = True

# 3状態: iwaparesu_mode=True(確定), op_confirmed_not_iwaparesu=True(否定確定), どちらもFalse(不明)
ogerpon_unnecessary = not iwaparesu_mode and _op_confirmed_not_iwaparesu
```

#### 各状態でのスコア

| 状態 | DISCARD | パーフェクトミキサー | PLAY/TO_BENCH | TO_HAND | ゼイユ |
|---|---|---|---|---|---|
| イワパレス確定 | -200,000 | -1 | 54,000/45,000（ベンチに出す） | 55,000 | undiscardable |
| 不明 | -200,000 | -1 | **-1（ベンチに出さない）** | 3,000 | undiscardable |
| イワパレスではない確定 | 60,000 | 80,000 | **-1（絶対に出さない）** | -10,000,000 | discardable |

**オーガポンをベンチに出すのはイワパレス確定時のみ。** 不明・否定確定ではPLAY/TO_BENCHともに-1。

### 対イワパレス戦

相手がイシズマイ(ID:344 or 532)またはイワパレス(ID:345)を場に出したら対イワパレスモードに入る。**一度検知したら試合中ずっと維持する。**

#### 検知条件

```python
# グローバル変数で試合中の状態を保持
_iwaparesu_mode = False

# ターン0でリセット
if state.turn == 0:
    _iwaparesu_mode = False

# 一度でも検知したら試合中ずっと維持
if not _iwaparesu_mode:
    for card in op_state.bench + op_state.active:
        if card.id in (344, 532, 345):  # イシズマイ, イワパレス
            _iwaparesu_mode = True
iwaparesu_mode = _iwaparesu_mode
```

#### 対イワパレスモードでの変更点

**ボスの指令の条件変更:**
- 通常: 相手exをワンパンできる場合のみ
- 対イワパレス: イシズマイがベンチにいれば呼び出す。いなければ他のポケモンを呼び出す（スコア93,000、パーフェクトミキサーの次に最優先）

**手貼り先の変更:**
- 相手バトル場がイワパレス → オーガポンに手貼り（エネ3枚まで）
- 相手バトル場がイワパレス以外 → ソウブレイズexに手貼り（通常通り）

| 対象 | 条件 | スコア |
|---|---|---|
| ベンチのオーガポン（エネ3枚未満） | 相手バトル場がイワパレス | 86,000 |
| バトル場のソウブレイズex | 相手バトル場がイワパレス以外 | 90,000（通常通り） |

**オーガポンのバトル場投入:**
- オーガポンに炎エネ3枚 AND 相手バトル場がイワパレス → にげるでオーガポンをバトル場に出す（スコア91,000）
- それ以外 → オーガポンはベンチで待機

**夜のタンカの制限:**
- 対イワパレスモードではオーガポンの回収専用。トラッシュにオーガポンがいる場合のみ使用
- 夜のタンカはハイパーボールのコスト等にしない（スコア-200,000）

**ディスカードの変更:**
- 夜のタンカ: スコア-200,000（絶対に捨てない）
- オーガポン: スコア-200,000（絶対に捨てない）
