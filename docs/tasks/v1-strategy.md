# V1 Step2: デッキの強い動きの言語化 — ソウブレイズexデッキ

## 勝利条件

相手のサイドを6枚取り切る。基本プランは **2-2-2**（exポケモンを3体倒す）。
しんえんほむらの青天井火力により、中盤以降はほぼ全てのポケモンを一撃で倒せる。

## デッキの核心：モグリュー + 闘エネルギーによるトラッシュ加速

従来のゼイユ・ハイパーボールによるトラッシュ肥やしに加え、**モグリューの特性「ほりまくり」**（ベンチに出したとき山札から基本闘エネルギーを3枚までトラッシュ）で一気にトラッシュを肥やす。ファイトゴングでモグリューや闘エネルギーをサーチできるため、序盤から安定してトラッシュにエネルギーを送り込める。

- **闘エネルギー11枚**: モグリューのほりまくりで山札から直接トラッシュに送る燃料
- **炎エネルギー7枚**: ソウブレイズexの攻撃コスト（炎1個）＋ゼイユ/ハイパーボールで手札からトラッシュ
- **ファイトゴング4枚**: モグリューまたは闘エネルギーをサーチ。序盤の安定性を大幅に向上

## ターン別の理想展開

### 先攻1ターン目

**目標: ベンチにカルボウ・モグリューを並べ、トラッシュにエネルギーを大量に送る**

1ターン目でベンチに最低1体はポケモンを出すことが最重要。バトル場のポケモンが倒された場合にベンチが空だと即負けになる。

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | なかよしポフィンでたねポケモンをベンチに出す（※） | 攻撃の起点 + トラッシュ肥やし。**ベンチに最低1体は必ず出す** |
| 最優先 | ファイトゴングでモグリューまたは闘エネルギーをサーチ（※） | モグリューが手札にない場合のサーチ手段。闘エネサーチ→ゼイユでトラッシュも有効 |
| 高 | ゼイユで手札のエネルギーをトラッシュし5枚ドロー | 先攻1ターン目から使える唯一のサポート。トラッシュ肥やしの主力 |
| 高 | ハイパーボールでソウブレイズexをサーチ（コストでエネルギーをトラッシュ） | 次ターンの進化準備 + トラッシュ肥やし |
| 中 | バトルコロシアムを出す | 相手のベンチ展開を制限する |
| 低 | カルボウで「ちからをあつめる」 | エネルギーを手札に集め、次ターンのゼイユ/ハイパーボールでまとめてトラッシュに送る布石 |

### 後攻1ターン目

**目標: スボミーでグッズロックしつつ、ベンチにカルボウを並べる**

理想盤面はバトル場にスボミー、ベンチにカルボウ。スボミーはすぐに倒されるため、ベンチにカルボウを出しておかないと次のターンに何もできなくなる。

なかよしポフィンの使い方（後攻1ターン目）：
- スボミーがまだいない → **カルボウ+スボミー**を出す
- スボミーがすでにいる → **カルボウを2体**出す
- **モグリューはポフィンで出さない**（ファイトゴング/ポケパッドでサーチする）

**スボミーが手札またはベンチにいて、バトル場に出せる状態なら、必ずスボミーで攻撃する。** グッズロックの価値は非常に高い。

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | パーフェクトミキサーで山札のエネルギーをトラッシュ | 手札にあるなら最初に打つ |
| 最優先 | なかよしポフィンでカルボウ+スボミー（またはカルボウ2体）をベンチに出す | グッズロック要員と攻撃の起点を同時に確保。**ベンチにカルボウを最低1体は必ず出す** |
| 最優先 | バトル場のポケモンにエネルギーを手貼りし、にげるでスボミーをバトル場に出す | スボミーがベンチにいる場合、手貼り→にげるでスボミーを前に出す。手貼りしたエネはにげるコストで消費されトラッシュへ |
| 高 | ファイトゴングでモグリューをサーチ → ベンチに出してほりまくり | グッズロックしながらトラッシュも肥やす |
| 高 | ゼイユ/リーリエの決心でドロー + トラッシュ肥やし | 手札を回しながらエネルギーをトラッシュへ |
| 高 | ハイパーボールでソウブレイズexをサーチ | 次ターンの進化準備 |
| 中 | バトルコロシアムを出す | 相手のベンチ展開を制限する |
| ターン最後 | スボミーで「むずむずかふん」 | 次の相手ターンでグッズが使えなくなる。**スボミーがバトル場に出せるなら必ず攻撃する** |

### 先攻2ターン目 / 後攻2ターン目

**目標: ソウブレイズexで攻撃を開始する**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | カルボウをソウブレイズexに進化させる | 攻撃開始の絶対条件 |
| 最優先 | ソウブレイズexに炎エネルギーを手貼り | しんえんほむらのコスト（炎1個） |
| 高 | まだモグリューを出していなければ、ファイトゴング/ポケパッドでモグリューを出してほりまくり（※） | トラッシュの闘エネを増やして火力底上げ |
| 高 | ゼイユで手札のエネルギーをトラッシュ | 火力を底上げ |
| 高 | ソウブレイズexがバトル場にいない場合、ポケモンいれかえで前に出す | 攻撃するにはバトル場にいる必要がある |
| 中 | 2体目のカルボウもベンチに残しておく | 1体目がきぜつした場合の保険 |
| 中 | バトルコロシアムを出す | 相手のベンチ展開を制限する |
| 攻撃 | しんえんほむら | モグリュー込みでトラッシュ8〜11枚なら190〜250ダメージ。多くのポケモンを一撃圏内 |

### 3ターン目以降

**目標: 高火力で相手のexポケモンを一撃で倒していく**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | ボスの指令で高価値ターゲットを引きずり出す | サイド2枚取れるexポケモンを優先的に狙う |
| 高 | ソウブレイズexがきぜつしていたら、次のソウブレイズexを準備して進化・エネ貼り・入れ替え | アタッカーの継続 |
| 高 | パーフェクトミキサー（手札にあるなら最初に打つ） | 山札からエネルギーを最大5枚トラッシュし火力を底上げ |
| 高 | キチキギスexの「さかてにとる」でドロー | きぜつ後のリカバリー。手札補充 |
| 中 | 2体目のモグリューを出してほりまくり（まだ出していなければ） | さらにトラッシュ肥やし |
| 中 | バトルコロシアムを出す | 相手のベンチ展開を制限する |
| 攻撃 | しんえんほむら | トラッシュ10枚以上で230ダメージ超。ほぼ全てのexを一撃 |

## サーチ系カードのポケモン選択優先度（※）

ファイトゴング・ハイパーボール等でたねポケモンを選ぶ際、場の状況に応じて優先度を切り替える。
**なかよしポフィンではカルボウとスボミーのみをサーチする（モグリューはサーチしない）。** モグリューはファイトゴングでサーチする。

| 優先 | 条件 | 出すポケモン | 理由 |
|---|---|---|---|
| 1 | 後攻1ターン目 | **スボミー**（最優先） | グッズロックが最も価値の高い行動。他のポケモンより優先して出す |
| 2 | 場のアタッカー（カルボウ+ソウブレイズex）が1体以下 | **カルボウ** | アタッカーラインの確保が最優先。進化元がいなければ攻撃できない |
| 3 | 場のアタッカーが2体で、カルボウが2枚以上いる | **モグリュー** | アタッカーラインは十分。ほりまくりでトラッシュ肥やしを加速する |
| - | 場のアタッカーが3体以上 | **出さない** | ベンチ枠をキチキギスex用に空けておく |

**重要: スボミーは後攻1ターン目以外では絶対にベンチに出さない。** ベンチ枠を圧迫し、サイドを献上するリスクしかない。

**重要: カルボウ+ソウブレイズexが場に3体以上いるならカルボウを追加で出さない。** ベンチ枠をキチキギスexのために空けておく。

### ポケモンがきぜつした後のサーチ優先度

ポケモンが倒された後、ハイパーボール・夜のタンカ等で次に出すポケモンの優先度：

| 優先 | 条件 | 出すポケモン | 理由 |
|---|---|---|---|
| 1 | 場にカルボウがいない | **カルボウ** | 進化元がいなければ次のアタッカーを立てられない |
| 2 | 場にカルボウはいるがソウブレイズexがいない | **ソウブレイズex** | 進化させてすぐにアタッカーとして使う |
| 3 | 場にソウブレイズexがいる（次のアタッカーが確保済み） | **キチキギスex** | アタッカーは足りているので、きぜつ後のドロー特性で手札を補充する |

### サポート

1ターンに1枚しか使えないため、状況に応じて最適なサポートを選択する。

#### サポート選択の優先度

| 優先 | 条件 | 使うサポート | 理由 |
|---|---|---|---|
| 1 | トラッシュのエネルギー枚数的に相手のexをワンパンできる | **ボスの指令** | サイド2枚取りのチャンス。倒せるなら最優先で狙う |
| 2 | 捨てられるカード > 捨てたくないカード | **ゼイユ** | エネルギーだけ捨てて5枚ドローできる |
| 3 | 上記いずれにも該当しない | **リーリエの決心** | 捨てたくないカードが多い手札では山札に戻して損失なく6枚ドロー |
| 4 | リーリエもゼイユもない | **ゼイユ**（最終フォールバック） | 条件が悪くてもサポートを使わないよりはマシ |

#### ドローソースの優先度

サポートに限らず、ドロー効果を持つ行動全体の優先度：

| 優先 | ドローソース | 理由 |
|---|---|---|
| 1 | キチキギスexの「さかてにとる」（特性） | サポート権を消費しない。きぜつ後に即ドロー |
| 2 | リーリエの決心（サポート） | 手札を山札に戻して6枚ドロー。通常時はゼイユより優先 |
| 3 | ゼイユ（サポート） | 手札をトラッシュして5枚ドロー。捨てられるカード > 捨てたくないカードの時のみリーリエより優先 |

#### ドロー系の使用タイミング

**ドロー系（サポート・特性・グッズ）は、今の手札でできる行動を一通り終えてから使う。** ただし、**手札に行動できるカードがなければ手札枚数に関係なくドローソースを使う。**

#### ゼイユの使用条件

ゼイユは手札を好きな枚数トラッシュして5枚ドローする。**捨てられるカードの枚数が捨てたくないカードの枚数より多ければ、リーリエより優先する。**

**捨てられるカード（ゼイユのコストにしてよいカード）:**
- 闘エネルギー
- 炎エネルギー（ただしトラッシュの炎エネルギーが4枚以上ならそれ以上は捨てない）
- スボミー（後攻1ターン目以外）

**捨てたくないカード:**
- ソウブレイズex
- カルボウ
- パーフェクトミキサー（ACE SPEC）
- サポートカード（ゼイユ自身を除く）

捨てられる枚数 > 捨てたくない枚数 → **ゼイユ**
捨てられる枚数 ≤ 捨てたくない枚数 → **リーリエの決心**

条件を満たさなくても、他にサポートがなければ最終フォールバックとしてゼイユを使う。

#### ゼイユでトラッシュするカードの優先度

| 優先 | カード | 条件 | 理由 |
|---|---|---|---|
| 1 | 闘エネルギー | 常に | トラッシュに送るのが最善 |
| 2 | スボミー | 後攻1ターン目以外 | 使わないカードを処理 |
| 3 | 炎エネルギー | トラッシュの炎エネが4枚未満 | トラッシュ肥やし。ただし場に炎エネ付きポケモンがいなければ1枚残す |
| 4 | 不要グッズ（序盤以降のなかよしポフィン等） | - | 腐ったカードを処理 |
| - | 炎エネルギー | トラッシュの炎エネが4枚以上 | **これ以上捨てない。** 手貼り用に手札に残す |
| - | ソウブレイズex・カルボウ・ACE SPEC・サポート | - | **トラッシュしない** |

| カード | いつ使うか | 注意点 |
|---|---|---|
| ゼイユ | 捨てられるカード > 捨てたくないカードの時（リーリエより優先） | 手札のエネルギーをトラッシュして5枚ドロー。炎エネはトラッシュに4枚以上あるならこれ以上捨てない |
| リーリエの決心 | 通常時のドローソース | 手札を全て山札に戻して6枚ドロー。エネルギーがトラッシュに行かないので注意 |
| ボスの指令 | 相手exをワンパンできる時 | 相手ベンチのexポケモンを引きずり出してサイド2枚取りを狙う |

### 夜のタンカの回収優先度

夜のタンカはトラッシュからポケモンまたは基本エネルギーを1枚回収する。**闘エネルギーは回収しない**（トラッシュに残して火力にする）。**ソウブレイズexも炎エネルギーも足りているなら使わない。**

**リソース枯渇チェック**を先に行い、該当しなければ**アタッカー準備のボトルネック**で判断する。

#### 1. リソース枯渇チェック（優先）

山札に必要なカードが残っているかを確認し、枯渇しているリソースを優先回収する。

| 条件 | 回収対象 | 理由 |
|---|---|---|
| トラッシュに炎エネルギーが5枚以上 | **炎エネルギー** | 山札にほぼ残っていない。回収しないと手貼りできなくなる |
| トラッシュにソウブレイズexが3体以上、かつ追加が必要 | **ソウブレイズex** | 山札・手札に残っていない。回収しないとアタッカーが立たない |

※ 「追加のソウブレイズexが必要」= `ceil(相手の残りサイド枚数 ÷ 2) > 場のソウブレイズex数`。場のソウブレイズexで十分なら回収不要。

#### 2. アタッカー準備のボトルネック（通常）

攻撃するには「場にカルボウ → ソウブレイズexに進化 → 炎エネルギーを付ける」の3ステップが必要。欠けているステップを補うカードを回収する。

| パターン | 場の状態 | 回収対象 | 理由 |
|---|---|---|---|
| A | 場にカルボウもソウブレイズexもいない | **カルボウ** | まず進化元がなければ始まらない |
| B | 場にカルボウはいるがソウブレイズexが手札にも場にもない | **ソウブレイズex** | 進化させてアタッカーにする |
| C | 場にソウブレイズexはいるが炎エネが付いておらず手札にもない | **炎エネルギー** | 手貼りしないと攻撃できない |
| - | ソウブレイズexも炎エネも足りている | **使わない** | 温存する |

### スタジアム

| カード | いつ使うか | 注意点 |
|---|---|---|
| バトルコロシアム | 相手スタジアムの上書き、またはベンチ制限が有利な場面 | お互いのベンチが4匹までになる。相手の展開を制限する |

## 行動の優先順序（ターン内）

**基本方針: 炎エネルギーの手貼りを最優先し、手札で行動できるカードを使い切ってからドローソースを使う。行動できるカードがなければ手札枚数に関係なくドローソースを使う。**

```
1. パーフェクトミキサー（手札にあるなら最初に打つ）
2. 炎エネルギー手貼り（どの行動よりも先に行う）
3. ベンチにアタッカーが育っているなら手貼り→にげるで前に出す
4. 進化（ソウブレイズexへの進化を最優先）
5. ポケモンをベンチに出す（モグリュー → ほりまくり発動）
6. グッズ使用（ファイトゴング → ポケパッド → なかよしポフィン → ハイパーボール → その他）
7. バトルコロシアムを出す（出し得）
8. ボスの指令（相手exをワンパンできる場合）
9. ドローソース（さかてにとる → リーリエ/ゼイユ）
10. 入れ替え（必要な場合）
11. 攻撃（しんえんほむら）← ターン最後の行動
```

※ 手札に行動できるカードがなければ、9のドローソースを2の直後（手貼りの次）に繰り上げる

## エネルギー手貼りの方針

しんえんほむらのコストは炎エネルギー1個のみ。**1体につきエネルギーは1枚だけ付ければよい。** すでにエネルギーが付いているポケモンには付けない。

### 手貼り先の優先度（エネルギーが0枚のポケモンのみ対象）

| 優先 | 対象 | 理由 |
|---|---|---|
| 1 | バトル場のソウブレイズex | 即しんえんほむらで攻撃可能にする |
| 2 | バトル場のカルボウ | ちからをあつめるで基本エネルギーを2枚サーチ |
| 3 | ベンチのソウブレイズex | 予備アタッカーの攻撃準備。倒された時にすぐ攻撃できる |
| 4 | ベンチのカルボウ | 進化後に即攻撃可能にする布石 |

以下のポケモンにはエネルギーを付けない：
- **モグリュー** — 攻撃しない。特性だけが重要
- **スボミー** — むずむずかふんのコストは無色1個で、にげるコスト分の手貼りで十分
- **キチキギスex** — 攻撃しない。特性だけが重要

### 例外1: 後攻1ターン目

バトル場のポケモン（カルボウ等）にエネルギーを手貼りし、にげるコストとして消費してスボミーをバトル場に出す。手貼りしたエネルギーはトラッシュに送られるため無駄にならない。

### 例外2: ベンチにエネルギー付きソウブレイズexがいる場合

バトル場のポケモンがソウブレイズex以外（カルボウ・モグリュー等）で、ベンチにエネルギーが付いたソウブレイズexがいる場合、**バトル場のポケモンにエネルギーを手貼りし、にげるでソウブレイズexを前に出す。** 手貼りしたエネルギーはにげるコストとしてトラッシュに送られるため無駄にならない。育ったアタッカーをすぐに攻撃に回す。

**この行動はパーフェクトミキサー・炎エネ手貼りに次ぐ高優先度で行う。** アタッカーが育っているのに前に出さないのは攻撃機会の損失になる。

## エネルギーのディスカード判定

しんえんほむらの火力はトラッシュのエネルギー枚数（炎・闘問わず）に直結するため、エネルギーは積極的にトラッシュに送りたい。
ただし、**炎エネルギーはソウブレイズexの攻撃コストとして必要**なため、最低1枚は手貼り用に確保する。

### 判定フロー（優先度順）

| 優先 | 状況 | エネを捨てる？ | 理由 |
|---|---|---|---|
| 1 | 闘エネルギーが手札にある | **捨てる** | 手貼りしても攻撃に使えない。トラッシュに送るのが最善 |
| 2 | このエネを捨てれば相手バトル場のポケモンを倒せる | **捨てる** | KOしてサイドを進めることが最優先 |
| 3 | ベンチにエネ付きソウブレイズexがいる | **捨てる** | 予備アタッカーがいるので、今のアタッカーが倒されても次ターン攻撃可能 |
| 4 | 予備アタッカーなし＆まだ手貼りしていない＆手札の炎エネが最後の1枚 | **残す** | 手貼り用の炎エネルギーを確保。これを捨てるとアタッカーにエネルギーを付けられない |
| 5 | それ以外 | **捨てる** | 手札に炎エネ2枚以上あるか、すでに手貼り済みなら捨ててOK |

### 「倒せるかどうか」の判定

- 現在のダメージ = `30 + トラッシュのエネルギー枚数 × 20`
- このエネを捨てた場合のダメージ = `30 + (トラッシュ枚数 + 1) × 20`
- 現在のダメージでは倒せないが、+1枚で倒せる → KO優先で捨てる

## パーフェクトミキサーの運用方針

パーフェクトミキサーは山札から好きなカードを5枚までトラッシュできるACE SPEC。**手札にあるなら他のどの行動よりも先に使う。**

### トラッシュ対象の優先度

| 優先 | カード | 条件 | 理由 |
|---|---|---|---|
| 1 | 闘エネルギー | 常に | できるだけ多くトラッシュする。しんえんほむらの火力に直結 |
| 2 | スボミー | 後攻1ターン目以外 | 使わないカード。山札から除去してサーチの邪魔を減らす |
| 3 | 炎エネルギー | バトル場+ベンチのポケモンに付いている炎エネルギーが合計3枚以上 | 攻撃コスト分は確保済み。余剰分はトラッシュして火力に変換 |

### 炎エネルギーをトラッシュする条件

バトル場とベンチのポケモンに付いている炎エネルギーの合計が3枚以上あること。手札の炎エネルギーは含めない。

これにより、現在のアタッカー + 予備アタッカー + さらにもう1体分の炎エネが場で確保できており、安全にトラッシュに回せる。

## スボミーの運用方針

- **後攻1ターン目**: スボミーをバトル場に出して「むずむずかふん」でグッズロック
- **後1の理想盤面**: バトル場にスボミー、ベンチにカルボウ
- **スボミーをバトル場に出す方法**: スボミーがベンチにいる場合、バトル場のポケモン（カルボウ等）にエネルギーを手貼りし、にげるでスボミーと入れ替える。手貼りしたエネルギーはにげるコストとしてトラッシュに送られるため無駄にならない
- **後1でスボミーがバトル場に出せるなら必ずむずむずかふんで攻撃する**: スボミーが手札またはベンチにいて、バトルポケモンと入れ替えられる状態なら、他の行動をすべて終えた後にスボミーで攻撃する。グッズロックの価値は非常に高い
- **スボミーがバトル場にいる限り入れ替えない**: 毎ターンむずむずかふんを打ち続けてグッズロックを維持する
- **スボミーが倒されたら**: ソウブレイズexが引き継いでしんえんほむらで攻撃を開始する
- スボミーが相手の攻撃を受けている間に、ベンチでソウブレイズexの準備（進化＋エネ貼り）とモグリューのほりまくりを進める

## モグリューの運用方針

- **バトルが始まる前（SETUP_BENCH_POKEMON）にはベンチに出さない**: ほりまくりの発動タイミングを自分のターンに合わせる
- **自分の最初の番で必ずベンチに出す**: 手札にモグリューがあれば最優先でベンチに出してほりまくりを発動する
- **ファイトゴング/ポケパッドでサーチする**: なかよしポフィンではサーチしない
- **2体出せれば最大6枚トラッシュ**: 2体目は2ターン目に出してもよい
- **ワザを使う必要はない**: 特性だけが重要。ベンチに置いたままでよい
- **ベンチ枠の管理**: バトルコロシアムでベンチ4枚制限の場合、カルボウ×2 + モグリュー + キチキギスex（または予備カルボウ）でベンチが埋まる点に注意

## ハイパーボールのコスト優先度

ハイパーボールは手札2枚をトラッシュしてポケモンをサーチする。コストに選ぶカードの優先度：

### コストにするカード（優先度順）

| 優先 | カード | 条件 | 理由 |
|---|---|---|---|
| 1 | 闘エネルギー | 常に | 手貼りしても攻撃に使えない。トラッシュに送るのが最善 |
| 2 | 炎エネルギー | 場に炎エネルギーが付いているポケモンがいる | すでに攻撃準備ができているので手札の炎エネはトラッシュへ |
| 3 | 炎エネルギー | 手札に炎エネが2枚以上ある | 1枚残せるのでもう1枚はトラッシュしてよい |
| 4 | 不要グッズ（序盤以降のなかよしポフィン等） | 使い道がない時 | 腐ったカードを処理 |

### コストにしてはいけないカード

| カード | 理由 |
|---|---|
| ソウブレイズex | メインアタッカー。手札にあっても進化先として必要 |
| カルボウ | ソウブレイズexの進化元。ベンチに出す必要がある |
| サポートカード全般 | 1ターン1枚の貴重なリソース。ドロー・妨害に必要 |
| パーフェクトミキサー（ACE SPEC） | 1枚限定の切り札。山札からエネルギーを直接トラッシュに送る最強手段 |

### 炎エネルギーを残す条件

**場に炎エネルギーが付いているポケモンがいない場合、手札の炎エネルギーは最低1枚残す。** これを捨てるとソウブレイズexにエネルギーを付けられず攻撃できなくなる。

## 特定状況への対応方針

### 自分のポケモンがきぜつした場合（全ポケモン共通）

スボミー・カルボウ・ソウブレイズex・モグリューのいずれがきぜつしても、以下の行動をとる：

1. キチキギスexの「さかてにとる」で3枚ドロー（特性が自動発動）
2. 夜のタンカでソウブレイズexまたはカルボウを回収
3. 次のソウブレイズexを立てて反撃

### トラッシュのエネルギーが足りない場合

1. パーフェクトミキサーで山札からエネルギーを最大5枚トラッシュ（最も効率的）
2. モグリューをベンチに出してほりまくりで闘エネ最大3枚トラッシュ
3. ファイトゴングで闘エネルギーをサーチ → ゼイユ/ハイパーボールでトラッシュ
4. ゼイユで手札のエネルギーをまとめてトラッシュ
5. ハイパーボールのコストでエネルギーをトラッシュ
6. カルボウの「ちからをあつめる」で山札からエネルギーを手札に集める → 次ターンでトラッシュに送る

### 対イワパレス戦（相手がイシズマイ/イワパレスを出してきた場合）

イワパレスは特性「しんぴのいしやど」によりexポケモンからのワザダメージを受けない。ソウブレイズexでは倒せないため、**非exのオーガポン かまどのめんで対処する。**

#### 基本方針

- **イワパレスを倒すのは最終手段。** できるだけイシズマイや他のポケモンをソウブレイズexで倒してサイドを進める
- **オーガポンは温存する。** エネルギーが3つ付いていて、かつ相手バトル場がイワパレスのときだけバトル場に出す
- **ボスの指令でイシズマイを呼び出してソウブレイズexで倒す** のが最も効率的
- **デッキ圧縮は引き続き重要。** パーフェクトミキサーやモグリューで闘エネをトラッシュする

#### 対イワパレス専用ムーブ

```
1. 相手がイシズマイ/イワパレスを出したことを検知
2. ベンチにオーガポン かまどのめんを出す
3. オーガポンに炎エネルギーを3枚付ける（毎ターン手貼りで3ターンかけて準備）
4. ボスの指令があればイシズマイを呼び出してソウブレイズexで倒す
5. イシズマイがいなければ他のベンチポケモンをソウブレイズexで倒す
6. 他に倒せるポケモンがいない場合のみ、オーガポンをバトル場に出してイワパレスを攻撃
```

#### オーガポンの運用ルール

| 条件 | 行動 |
|---|---|
| オーガポンに炎エネが3枚付いている AND 相手バトル場がイワパレス | バトル場に出して「やきこがす」で攻撃（80ダメージ + やけど。イワパレスは草タイプで炎弱点なので160ダメージ。イワパレスHP150を一撃） |
| オーガポンに炎エネが3枚未満 | ベンチで待機。ただし手貼りはソウブレイズexを優先し、ソウブレイズexにエネが付いていればオーガポンに付ける |
| 相手バトル場がイワパレスではない | オーガポンは出さない。ソウブレイズexで攻撃。手貼りもソウブレイズexを優先 |
| オーガポンがきぜつした | 夜のタンカで回収する |

#### 夜のタンカの扱い（対イワパレス時）

- **夜のタンカはオーガポンの回収専用に温存する。** 他の目的では使わない
- **夜のタンカは絶対にトラッシュしない（ハイパーボールのコスト等にしない）**
- トラッシュにオーガポンがいる場合のみ夜のタンカを使用する

#### 対イワパレス時の行動順序

```
1. パーフェクトミキサー（手札にあるなら最初に打つ）
2. ボスの指令（最優先。イシズマイや他のポケモンを呼び出す）
3. 炎エネルギー手貼り（相手バトル場による）
   - 相手バトル場がイワパレス → オーガポンに手貼り
   - 相手バトル場がイワパレス以外 → ソウブレイズexに手貼り
4. 進化・ベンチ展開・グッズ使用
5. 攻撃
   - 相手バトル場がイワパレス AND オーガポンに炎エネが3枚付いている → オーガポンをバトル場に出してやきこがす
   - それ以外 → ソウブレイズexでしんえんほむら
```

#### ボスの指令の使い方（対イワパレス時）

| 相手ベンチの状態 | ボスの指令で呼ぶポケモン |
|---|---|
| イシズマイがいる | イシズマイを呼び出し、ソウブレイズexで倒す（進化前に潰す） |
| イシズマイがいない | 他のポケモンを呼び出し、ソウブレイズexで倒す |

### 相手が高HPポケモン（HP300以上）を出してきた場合

- トラッシュに14枚以上のエネルギーが必要（310ダメージ）
- モグリュー2体のほりまくり（最大6枚）+ ゼイユ/ハイパーボールで十分到達可能
- 無理に一撃を狙わず、2回攻撃で倒すことも視野に入れる
- ボスの指令でHP低めのベンチポケモンを狙う選択肢もある
