# V1 Step2: デッキの強い動きの言語化 — ソウブレイズexデッキ

## 勝利条件

相手のサイドを6枚取り切る。基本プランは **2-2-2**（exポケモンを3体倒す）。
しんえんほむらの青天井火力により、中盤以降はほぼ全てのポケモンを一撃で倒せる。

## ターン別の理想展開

### 先攻1ターン目

**目標: カルボウを複数並べ、トラッシュにエネルギーを送り始める**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | なかよしポフィンでカルボウ2枚をベンチに出す | 攻撃の起点。カルボウがいなければ何も始まらない |
| 高 | ゼイユで手札のエネルギーをトラッシュし5枚ドロー | 先攻1ターン目から使える唯一のサポート。トラッシュ肥やしの主力 |
| 高 | ハイパーボールでソウブレイズexをサーチ（コストでエネルギーをトラッシュ） | 次ターンの進化準備 + トラッシュ肥やし |
| 中 | ロケット団の監視塔を出す | 相手の無色ポケモンの特性を封じる。先出しで相手の初動を妨害 |
| 低 | カルボウで「ちからをあつめる」 | エネルギーを手札に集め、次ターンのゼイユ/ハイパーボールでまとめてトラッシュに送る布石 |

### 後攻1ターン目

**目標: スボミーでグッズロックして相手の展開を1ターン遅らせる**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | なかよしポフィンでスボミー+カルボウをベンチに出す | グッズロック要員と攻撃の起点を同時に確保 |
| 最優先 | スボミーをバトル場に出して「むずむずかふん」 | 次の相手ターンでグッズが使えなくなる。相手の展開を大幅に遅延させる |
| 高 | ゼイユ/リーリエの決心でドロー + トラッシュ肥やし | 手札を回しながらエネルギーをトラッシュへ |
| 高 | ハイパーボールでソウブレイズexをサーチ | 次ターンの進化準備 |

### 先攻2ターン目 / 後攻2ターン目

**目標: ソウブレイズexで攻撃を開始する**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | カルボウをソウブレイズexに進化させる | 攻撃開始の絶対条件 |
| 最優先 | ソウブレイズexに炎エネルギーを手貼り | しんえんほむらのコスト（炎1個） |
| 高 | ゼイユ/アオキの手際で手札のエネルギーをトラッシュ | 火力を底上げ |
| 高 | ソウブレイズexがバトル場にいない場合、ポケモンいれかえで前に出す | 攻撃するにはバトル場にいる必要がある |
| 中 | 2体目のカルボウもベンチに残しておく | 1体目がきぜつした場合の保険 |
| 攻撃 | しんえんほむら | トラッシュ5〜8枚で130〜190ダメージ。たねポケモンや非exは倒せる圏内 |

### 3ターン目以降

**目標: 高火力で相手のexポケモンを一撃で倒していく**

| 優先度 | 行動 | 理由 |
|---|---|---|
| 最優先 | ボスの指令で高価値ターゲットを引きずり出す | サイド2枚取れるexポケモンを優先的に狙う |
| 高 | ソウブレイズexがきぜつしていたら、次のソウブレイズexを準備して進化・エネ貼り・入れ替え | アタッカーの継続 |
| 高 | アンフェアスタンプ（きぜつ直後に使用） | 相手の手札を2枚にリセットし、立て直しを困難にする |
| 高 | キチキギスexの「さかてにとる」でドロー | きぜつ後のリカバリー。手札補充 |
| 攻撃 | しんえんほむら | トラッシュ10枚以上で230ダメージ超。ほぼ全てのexを一撃 |

## カードごとの使いどころ

### ポケモン

| カード | いつ使うか | 注意点 |
|---|---|---|
| カルボウ | 1ターン目にベンチに2〜3枚並べる | なかよしポフィンで展開。手貼りで「ちからをあつめる」も有効 |
| ソウブレイズex | 2ターン目以降メインアタッカー | 常にバトル場に1体、ベンチに予備を準備 |
| スボミー | 後攻1ターン目以降、倒されるまで | スボミーがバトル場にいる限りむずむずかふんを打ち続ける。入れ替えない。倒されたらソウブレイズexにバトンタッチ |
| キチキギスex | 中盤以降 | 自分のポケモンがきぜつした後に特性でドロー。ベンチに置くタイミングを見極める |

### グッズ

| カード | いつ使うか | 注意点 |
|---|---|---|
| なかよしポフィン | 1ターン目 | カルボウ(HP70)とスボミー(HP70以下)を展開。序盤以降は腐る |
| ハイパーボール | 毎ターン | コストでエネルギーを捨てることを意識。ソウブレイズex/キチキギスexのサーチに |
| 夜のタンカ | 中盤以降 | ソウブレイズexがきぜつした後の回収。エネルギー回収には基本使わない（トラッシュに残したい） |
| ポケモンいれかえ | 2ターン目、または必要時 | スボミーやカルボウからソウブレイズexに交代する |
| アンフェアスタンプ | 自分のポケモンきぜつ直後 | 1枚限定（ACE SPEC）。相手のサイドが少ない終盤で使うと効果的 |
| クラッシュハンマー | いつでも | コイン依存だが、相手のエネルギーを剥がして攻撃を遅延させる |

### サポート

| カード | いつ使うか | 注意点 |
|---|---|---|
| ゼイユ | 1ターン目最優先 | 先攻1ターン目から使える。手札のエネルギーをトラッシュして5枚ドロー |
| リーリエの決心 | 手札が少ない時 | 手札を全て山札に戻して6枚ドロー。エネルギーがトラッシュに行かないので注意 |
| アオキの手際 | 手札にエネルギーが多い時 | 手札を全てトラッシュ → ポケモン・サポート・基本エネ各1枚サーチ。手札のエネルギーを一気にトラッシュへ |
| ボスの指令 | 3ターン目以降 | 相手ベンチのexポケモンを引きずり出してサイド2枚取りを狙う |

### スタジアム

| カード | いつ使うか | 注意点 |
|---|---|---|
| ロケット団の監視塔 | 1ターン目、または相手スタジアムの上書き | 無色ポケモンの特性を無効化。相手のドロー系特性持ちを封じる |

## 行動の優先順序（ターン内）

サンプルコードと同様に、1ターン内の行動は以下の順序で処理する：

```
1. 進化（ソウブレイズexへの進化を最優先）
2. グッズ使用（なかよしポフィン → ハイパーボール → その他）
3. サポート使用（ゼイユ/アオキの手際/リーリエの決心/ボスの指令）
4. エネルギー手貼り（ソウブレイズexに優先）
5. 入れ替え（必要な場合）
6. 攻撃（しんえんほむら）← ターン最後の行動
```

## エネルギーのディスカード判定

しんえんほむらの火力はトラッシュのエネルギー枚数に直結するため、エネルギーは積極的にトラッシュに送りたい。
ただし、アタッカーにエネルギーが付いていない状況で全部捨てると攻撃できなくなる。

### 判定フロー（優先度順）

| 優先 | 状況 | エネを捨てる？ | 理由 |
|---|---|---|---|
| 1 | このエネを捨てれば相手バトル場のポケモンを倒せる | **捨てる** | KOしてサイドを進めることが最優先 |
| 2 | ベンチにエネ付きソウブレイズexがいる | **捨てる** | 予備アタッカーがいるので、今のアタッカーが倒されても次ターン攻撃可能 |
| 3 | 予備アタッカーなし＆まだ手貼りしていない＆最後の1枚 | **残す** | 手貼り用のエネルギーを確保。これを捨てるとアタッカーにエネルギーを付けられない |
| 4 | それ以外 | **捨てる** | 手札に2枚以上あるか、すでに手貼り済みなら捨ててOK |

### 「倒せるかどうか」の判定

- 現在のダメージ = `30 + トラッシュのエネルギー枚数 × 20`
- このエネを捨てた場合のダメージ = `30 + (トラッシュ枚数 + 1) × 20`
- 現在のダメージでは倒せないが、+1枚で倒せる → KO優先で捨てる

## スボミーの運用方針

- **後攻1ターン目**: スボミーをバトル場に出して「むずむずかふん」でグッズロック
- **スボミーがバトル場にいる限り入れ替えない**: 毎ターンむずむずかふんを打ち続けてグッズロックを維持する
- **スボミーが倒されたら**: ソウブレイズexが引き継いでしんえんほむらで攻撃を開始する
- スボミーが相手の攻撃を受けている間に、ベンチでソウブレイズexの準備（進化＋エネ貼り）を進める

## ディスカードで捨ててはいけないカード

ハイパーボールのコスト等でカードを捨てる際、以下のカードは絶対に捨てない。

| カード種別 | 理由 |
|---|---|
| ポケモンカード全般 | アタッカー・ベンチ要員として必要。捨てると盤面構築ができなくなる |
| アンフェアスタンプ（ACE SPEC） | 1枚限定の切り札。きぜつ後の逆転カードとして温存する |
| クラッシュハンマー | 相手のエネルギーを剥がす妨害カード。捨てずに使い切る |

ディスカード時の優先度：**エネルギー（最優先で捨てる）> 不要グッズ/サポート > それ以外 > ポケモン・ACE SPEC（捨てない）**

## 特定状況への対応方針

### 自分のポケモンがきぜつした場合（全ポケモン共通）

スボミー・カルボウ・ソウブレイズexのいずれがきぜつしても、以下の行動をとる：

1. キチキギスexの「さかてにとる」で3枚ドロー（特性が自動発動）
2. アンフェアスタンプが手札にあれば使用（相手手札2枚リセット）
3. 夜のタンカでソウブレイズexまたはカルボウを回収
4. 次のソウブレイズexを立てて反撃

### トラッシュのエネルギーが足りない場合

1. ゼイユ・アオキの手際で手札のエネルギーをまとめてトラッシュ
2. ハイパーボールのコストでエネルギーをトラッシュ
3. カルボウの「ちからをあつめる」で山札からエネルギーを手札に集める → 次ターンでトラッシュに送る

### 相手が高HPポケモン（HP300以上）を出してきた場合

- トラッシュに14枚以上のエネルギーが必要（310ダメージ）
- 無理に一撃を狙わず、2回攻撃で倒すことも視野に入れる
- ボスの指令でHP低めのベンチポケモンを狙う選択肢もある
